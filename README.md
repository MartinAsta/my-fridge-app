# What's in My Fridge?

A full-stack web application that helps users track the ingredients they have at home and discover what meals they can cook with them.

## Concept

Users can:
- Add ingredients to their personal "fridge"
- Create and store custom recipes (as ingredient lists)
- View all meals they can currently make based on available ingredients
- Consume ingredients when a meal is cooked, keeping the fridge up to date

The goal is to reduce food waste and make cooking decisions easier.

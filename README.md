# My Fridge App

My Fridge App is a restaurant management application.

It lets users create and manage restaurants, ingredients, fridge stock, dishes, orders, and cash registers.

Features include:
- restaurant ownership and staff roles
- ingredient management
- fridge stock tracking
- dish creation with ingredient requirements
- menu viewing for staff
- order creation by waiters
- order logs for restaurant owners
- cash register updates when ingredients are bought or orders are placed
